# ML-Internship-Task
# Task 1: Exploring and Visualizing Iris Dataset

## Objective
The objective of this task is to learn how to load, inspect, and visualize a dataset using Python libraries like Pandas, Matplotlib, and Seaborn.

## Dataset Used
- Iris Dataset (CSV format from GitHub)
- Features include:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
  - Species

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed

### 1. Data Loading
- Dataset loaded using Pandas (`read_csv`)

### 2. Data Inspection
- Used `.head()` to view first rows
- Used `.shape` to check dataset size
- Used `.info()` and `.describe()` for summary statistics

### 3. Data Visualization
- Scatter plot created to show relationship between features
- Histograms used to show distribution of values
- Box plots used to detect outliers

## Key Observations
- Different iris species show clear differences in petal measurements
- Some features are more useful for classification than others
- No major missing values in dataset

## Conclusion
This task helped in understanding basic data exploration and visualization techniques which are important steps in any Machine Learning project.
