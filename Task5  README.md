# Task 5: Mental Health Support Chatbot (Fine-Tuned)

## 🎯 Objective
To build an AI chatbot that provides empathetic and supportive responses for mental health-related feelings such as stress, anxiety, and sadness.

## Project Idea
This chatbot is trained to understand emotional input from users and respond in a caring and supportive tone.

##  Dataset
A small custom emotional dataset was created for training, containing examples of:
- Stress-related responses
- Anxiety support replies
- Sadness and emotional comfort messages

##  Model Used
GPT-2 (Hugging Face)

## Approach
1. Created a small emotional dataset
2. Converted dataset into Hugging Face format
3. Tokenized text data
4. Fine-tuned GPT-2 using Trainer API
5. Generated empathetic responses using trained model

## Features
- Emotion-aware responses
- Supportive and friendly tone
- Safe conversational AI behavior
- Simple mental health support chatbot

## Tools & Libraries
- Python
- Hugging Face Transformers
- Hugging Face Datasets
- PyTorch

##  Example Usage

Input:
"I feel very stressed and anxious"

Output:
A calm and supportive response generated by the model
