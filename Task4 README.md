# Task 4: Health Chatbot (No API)

## Objective
Build a simple health chatbot using NLP model.

## Model Used
GPT-2 (Hugging Face Transformers)

## Features
- Answers general health questions
- Safe and friendly responses
- No API required

## Tools
Python, Transformers, Hugging Face

## Note
This chatbot gives general information only, not medical advice.
