# Task 3: Heart Disease Prediction

## Objective
Build a machine learning model to predict heart disease.

## Dataset
Heart Disease 

## Steps Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Label encoding for categorical data
- Logistic Regression model training
- Model evaluation using accuracy, confusion matrix, and ROC curve
- Feature importance analysis

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Scikit-learn

## Results
Model achieved good accuracy and identified key health features affecting heart disease prediction.
